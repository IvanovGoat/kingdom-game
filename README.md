# Kingdom Game

Консольная экономическая стратегия на C++ для курсового проекта.

Игрок управляет государством в течение 10 ходов, распределяя ресурсы, строя инфраструктуру, обучая армию и воюя с врагами. В конце каждого дня происходит случайное событие.

## Функционал

- Покупка еды за золото
- Строительство (увеличивает доход от инфраструктуры)
- Мероприятия для народа (повышают счастье)
- Тренировка бойцов (увеличивает армию)
- Война (случайный противник или фиксированная сила)
- Случайные события в конце дня
- Итоговая статистика с качественной оценкой правления
- Сохранение статистики в файл

## Требования

- C++17 или новее
- CMake 3.14+
- Компилятор с поддержкой стандарта (MSVC, GCC, Clang)
